## Sobre o projeto
O **EyesOnRoad** é uma aplicação Fullstack desenvolvida para monitorar a sonolência de motoristas ao volante. O repositório reúne todas as etapas, códigos e documentações produzidas durante o desenvolvimento do sistema.

O Projeto de Conclusão de Curso foi desenvolvido com os conhecimentos adquiridos nas disciplinas do Curso Técnico de Desenvolvimento de Sistemas do **SENAI Sorocaba - Gaspar Ricardo Júnior**, sob a orientação do Prof. Vedilson Prado e Prof. Luciana Fugita. O objetivo é auxiliar na diminuição de acidentes por sonolência ao dirigir

---

## Tecnologias Utilizadas

### Backend
**Java:** Linguagem base do sistema.
**Spring Boot:** Framework para construção da API REST.
**Python:** Linguagem utilizada para Detecção da sonolência

### Banco de Dados
**MySQL:** Gerenciamento e persistência dos dados.

### Frontend
**HTML5 e CSS3:** Estrutura e estilização da interface.
**JavaScript:** Lógica de consumo da API e interatividade.
